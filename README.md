# MASICS Mario Cloud Viewer

This repository contains the mobile-friendly MASICS legal evidence review viewer.

## Current status

The mobile layout was reviewed and user-tested on July 11, 2026. Dropbox sign-in, queue loading, evidence access, scrolling, sticky review controls, Decision, Notes, and Save Online remained available in the accepted layout.

See [`docs/MOBILE_LAYOUT_REVIEW_2026-07-11.md`](docs/MOBILE_LAYOUT_REVIEW_2026-07-11.md) for the completed review, stable-baseline notes, and the exact scope of the mobile changes.
